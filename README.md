# Coding-challenge

## Work Environment
- Framework Flask 1.1.2
- Python 3.8.5

## Usage
- Create & activaie virtual environment.
- Install Flask
```bash
pip install Flask
```
- Install requests
```bash
pip install requests
```
- Clone the project
- Start the server using the command 
```bash
flask run
```
